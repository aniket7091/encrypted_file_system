How Verification Works

1

Encrypted File Hash Check

Computes SHA-256 hash of the encrypted file and compares with stored hash to detect tampering

2

Decryption

Decrypts the file using RSA-protected symmetric key

3

Original File Hash Check

Computes SHA-256 hash of decrypted data and compares with original file hash

4

Verification Result

Reports success if all checks pass, or failure if tampering is detected

ℹ️ What Gets Verified

  • Encrypted File Integrity: Ensures the encrypted file hasn't been modified
  • Decryption Correctness: Verifies the file can be decrypted successfully
  • Original Data Integrity: Confirms decrypted data matches the original file
  • Tamper Detection: Identifies any unauthorized modifications
